    # 공지 사항

    • 성남시립도서관 앱 느려진 문제 조치
    • 모바일에서 전자책, 오디오북, 전자잡지 이용방법
    • 감성 태교교실 수강생 모집
    • 성남시 공공도서관 희망도서 신청권수 변경(10.1~)
    • 2019년도 전자책 희망도서 신청 마감(~10.15까지)
    • 임산부와 가족을 위한 감성 태교교실
    • 손글씨 캘리그라피 운영
    • 2019년 하반기 시민정보활용교육
